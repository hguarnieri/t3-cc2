Local dos casos de Teste: /src/trab3/casosDeTeste/

Para executar o projeto:

1) No terminal, vá até a pasta dist
2) Execute o seguinte comando:  java -jar Trab1.jar ../src/trab3/casosDeTeste/{tipo_caso_teste}/{nome_do_arquivo}
2.1) Caso seja do tipo sintático ou semântico, os erros serão exibidos no terminal
2.2) Caso seja do tipo corretos, o arquivo gerado.html será criado na pasta correspondente:
  casosDeTeste/resultado_gerado/sites/CV, se o caso de teste for de CV
  casosDeTeste/resultado_gerado/sites/Blog, se o caso de teste for de Blog
3) Abra o arquivo gerado.html no navegador e veja o resultado!
